
# Магазин кухонных ножей — ретема проекта

Это ретема исходного Django‑проекта «Магазин кухонных ножей» под тему «магазин **кухонных ножей**». 
Основная логика и модели сохранены, но визуал и терминология в интерфейсе изменены.

## Что сделано
- Новый визуальный стиль **EdgeSteel** (тёмная тема, новые шрифты, карточки товаров, кнопки).
- Изменены тексты в шаблонах: «Ножи» → «Ножи», «Бренд» → «Бренды», «Серия» → «Серии».
- В `knifestore/models.py` изменён путь загрузки изображений: `knifes/covers/` → `knives/covers/` и поле «Обложка Ножи» → «Фото ножа».
- Обновлён заголовок и навигация в `knifestore/templates/base.html`.
- Обновлён каталог в `knifestore/templates/store/knife_store.html`.

> Обратите внимание: названия переменных и URL оставлены прежними (например, `knife`, `knife_list`) для минимального вмешательства в код. Это не видно пользователю.

## Как запустить
1. Установите зависимости (если используются): `pip install -r requirements.txt`
2. Примените миграции: `python manage.py migrate`
3. Создайте суперпользователя (при необходимости): `python manage.py createsuperuser`
4. Запустите сервер: `python manage.py runserver`
5. Откройте в браузере: http://127.0.0.1:8000/

## Где править тексты/вид
- Главный шаблон: `knifestore/templates/base.html`
- Каталог: `knifestore/templates/store/knife_store.html`
- Общие стили: `static/css/main.css` (раздел **EdgeSteel theme override**)

## Что можно улучшить дополнительно
- Переименовать классы/модели (`knife` → `Knife`, `Publisher` → `Brand`) и соответствующие URL/вьюхи.
- Добавить новые поля: сталь, длина лезвия, назначение (шефский, овощной и т.д.).
- Загрузить новые изображения в `media/knives/covers/`.
- Перегенерировать фикстуры данных под тематику ножей.
